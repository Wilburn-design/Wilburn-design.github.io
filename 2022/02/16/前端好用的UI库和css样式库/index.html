前端好用的UI库和css样式库

发表于 2022-02-16 更新于 2022-07-06 分类于 前端 ， UI库 ， css库

Web端和移动端我常用的库如下。

一、Web端

  • ElementUI

之前在开发Vue2.x的时候使用的ElementUI，样式简介好看，支持vue2.x，各种组件的属性以及API方法较为完善。

详情见：ElementUI

  • ElementPlus

使用Vue3.2和TypeScript进行开发，支持setup语法糖格式，提供完整的类型定义文件。并使用 Composition API 降低耦合，简化逻辑。并在原来的UI基础上进行了体系化的改进。

详情见：ElementPlus

  • WindiCSS

对使用过Tailwind的开发者来说，WindiCSS就可以说是Tailwind的替代，完美兼容Tailwind v2.0。我认为在这里几乎包含了开发者能用到的大部分CSS库，而且方便快速，与Element系列可以联用，兼容性好。

二、移动端

  • Vant

之前在给学校做实验室移动应用小程序时使用的Vant + Uniapp，觉得VantUI这个框架足够轻量，平均组件体积小于1KB，提供的UI样式也很多且稳定，有兴趣的兄弟可以使用试试。

详情见：Vant Weapp

  • ColorUI

与WindiCSS一样，ColorUI是一个css类库，视觉效果非常好看，在众多框架中脱颖而出，与Vant可以联用。

详情见：ColorUI
